# Android

Android application scaffold for bloop.

## Included

- Gradle-based Android app project under `android/`
- Jetpack Compose application shell
- Minimal launcher activity and manifest
- Theme and placeholder home screen

## Open In Android Studio

Open the `android/` directory in Android Studio and allow it to sync with the embedded JDK.

If Android Studio asks to install missing SDK components, install them there. This scaffold targets:

- Compile SDK `35`
- Target SDK `35`
- Min SDK `26`
